If you want to create a document-author service that will run in every room:
```
meshagent service create --name "document-author" --image docker.io/meshagent/sample-document-author:v0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=document-author"
```

If you want to test the document-author with a single room:
```
meshagent service test --room my-room --name "document-author" --image docker.io/meshagent/sample-document-author:v0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=document-author"
```
