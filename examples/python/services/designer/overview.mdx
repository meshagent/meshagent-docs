If you want to create a designer service that will run in every room:
```
meshagent service create --name "designer" --image docker.io/meshagent/sample-designer:0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=designer"
```

If you want to test the designer with a single room:
```
meshagent service test --room my-room --name "designer" --image docker.io/meshagent/sample-designer:0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=designer"
```
