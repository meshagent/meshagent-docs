## Overview

The `SecretsClient` is the Room API for room-scoped secrets and participant-scoped credential exchange. Use it to store secrets, retrieve them later, and handle delegated secret or OAuth flows between room participants.

## Why use the Secrets API?

- Keep credentials out of prompts, code, and static configuration.
- Share or delegate secrets between participants without exposing raw values broadly.
- Handle OAuth authorization flows inside the same room where the agent or service will use the resulting token.

## How it works

Secrets are stored by ID and can include metadata such as a name, MIME type, or delegation target. In addition to basic CRUD, the Secrets API supports request/provide/reject flows so one participant can ask another participant for a secret or OAuth authorization.

## Permissions and grants

Secrets access is more sensitive than most other Room APIs. Deployed services need the appropriate secrets-related Room API grants, and delegation flows still respect room identity and authorization boundaries. See [API Scopes](../rest_api/api_scopes) and [Packaging and Deploying Services](../services_room_containers/packaging_and_deploying/packaging).

## CLI and SDK availability

- **CLI**: room-scoped secret commands are available under `meshagent room secret ...`.
- **Python and Dart**: full secret and OAuth helpers.
- **TypeScript and .NET**: core secret CRUD helpers.

## API Methods

### list_secrets
- **Description**: List secrets saved to the current room.
- **Returns**: `list[SecretInfo]` (`id`, `type`, `name`, `delegated_to`).

<CodeGroup>
```bash CLI
meshagent room secret list --room=myroom
```
```python Python
secrets = await room.secrets.list_secrets()
for s in secrets:
    print(s.id, s.type, s.name, s.delegated_to)
```
```javascript NodeJs
const secrets = await room.secrets.listSecrets();
for (const s of secrets) {
  console.log(s.id, s.type, s.name, s.delegatedTo);
}
```
```typescript TypeScript
const secrets = await room.secrets.listSecrets();
for (const s of secrets) {
  console.log(s.id, s.type, s.name, s.delegatedTo);
}
```
```dart Dart
final secrets = await room.secrets.listSecrets();
for (final s in secrets) {
  print('${s.id} ${s.type} ${s.name} ${s.delegatedTo}');
}
```
```dotnet C#
var secrets = await room.Secrets.ListSecrets();
foreach (var s in secrets)
{
  Console.WriteLine($"{s.Id} {s.Type} {s.Name} {s.DelegatedTo}");
}
```
</CodeGroup>

### set_secret
- **Description**: Save or update a secret value.
- **Parameters**:
  - Python: `secret_id|type|name`, `data`, `delegated_to`, `for_identity`.
  - JS/TS/Dart/.NET: `secretId`, `data`, optional metadata (`type`/`mimeType`, `name`, delegation fields).

<CodeGroup>
```bash CLI
meshagent room secret set \
  --room myroom \
  --id my-secret \
  --text 'secret-value' \
  --type text/plain
```
```python Python
await room.secrets.set_secret(
    secret_id="my-secret",
    type="text/plain",
    data=b"secret-value",
)
```
```javascript NodeJs
await room.secrets.setSecret({
  secretId: "my-secret",
  data: new TextEncoder().encode("secret-value"),
  mimeType: "text/plain",
});
```
```typescript TypeScript
await room.secrets.setSecret({
  secretId: "my-secret",
  data: new TextEncoder().encode("secret-value"),
  mimeType: "text/plain",
});
```
```dart Dart
await room.secrets.setSecret(
  secretId: "my-secret",
  data: Uint8List.fromList("secret-value".codeUnits),
  mimeType: "text/plain",
);
```
```dotnet C#
await room.Secrets.SetSecret(
  secretId: "my-secret",
  data: System.Text.Encoding.UTF8.GetBytes("secret-value"),
  type: "text/plain"
);
```
</CodeGroup>

### get_secret
- **Description**: Fetch a stored secret by ID.
- **Returns**: Secret bytes (`FileContent`/`FileChunk`) or `None`/`null` when missing.

<CodeGroup>
```bash CLI
meshagent room secret get \
  --room myroom \
  --id my-secret
```
```python Python
secret = await room.secrets.get_secret(secret_id="my-secret")
if secret is not None:
    print(secret.data)
```
```javascript NodeJs
const secret = await room.secrets.getSecret({ secretId: "my-secret" });
if (secret) {
  console.log(secret.data);
}
```
```typescript TypeScript
const secret = await room.secrets.getSecret({ secretId: "my-secret" });
if (secret) {
  console.log(secret.data);
}
```
```dart Dart
final secret = await room.secrets.getSecret(secretId: "my-secret");
if (secret != null) {
  print(secret.data);
}
```
```dotnet C#
var secret = await room.Secrets.GetSecret("my-secret");
if (secret != null)
{
  Console.WriteLine(secret.Data.Length);
}
```
</CodeGroup>

### request_oauth_token
- **Description**: Start an OAuth flow and return an access token.
- **Availability**: Python and Dart.

<CodeGroup>
```bash CLI
meshagent room secret oauth \
  --room myroom \
  --from-participant-id participant_123 \
  --client-id client-id \
  --authorization-endpoint https://accounts.example.com/o/oauth2/v2/auth \
  --token-endpoint https://oauth2.example.com/token \
  --redirect-uri https://app.example.com/oauth/callback \
  --scopes scope1
```
```python Python
from meshagent.api.oauth import OAuthClientConfig

token = await room.secrets.request_oauth_token(
    from_participant_id="participant_123",
    redirect_uri="https://app.example.com/oauth/callback",
    oauth=OAuthClientConfig(
        client_id="client-id",
        authorization_endpoint="https://accounts.example.com/o/oauth2/v2/auth",
        token_endpoint="https://oauth2.example.com/token",
        scopes=["scope1"],
    ),
)
print(token)
```
```dart Dart
final token = await room.secrets.requestOAuthToken(
  fromParticipantId: "participant_123",
  redirectUri: Uri.parse("https://app.example.com/oauth/callback"),
  oauth: OAuthClientConfig(
    clientId: "client-id",
    authorizationEndpoint: "https://accounts.example.com/o/oauth2/v2/auth",
    tokenEndpoint: "https://oauth2.example.com/token",
    scopes: ["scope1"],
  ),
  delegateTo: "participant_123",
);
print(token);
```
</CodeGroup>

### request_secret / provide_secret / reject_secret
- **Description**: Request a secret from another participant, then provide or reject it.
- **Availability**: Python and Dart.

<CodeGroup>
```bash CLI
meshagent room secret request \
  --room myroom \
  --from-participant-id participant_123 \
  --url secret://example \
  --type application/octet-stream
```
```python Python
secret = await room.secrets.request_secret(
    from_participant_id="participant_123",
    url="secret://example",
    type="application/octet-stream",
)
print(secret)
```
```dart Dart
final secret = await room.secrets.requestSecret(
  fromParticipantId: "participant_123",
  url: "secret://example",
  type: "application/octet-stream",
);
print(secret);
```
</CodeGroup>

---

### provide_oauth_authorization / reject_oauth_authorization
- **Description**: Complete or reject a pending OAuth authorization request.
- **Availability**: Python and Dart.
