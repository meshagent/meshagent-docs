If you want to create a presentation-author service that will run in every room:
```
meshagent service create --name "presentation-author" --image docker.io/meshagent/sample-presentation-author:v0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=presentation-author"
```

If you want to test the presentation-author with a single room:
```
meshagent service test --room my-room --name "presentation-author" --image docker.io/meshagent/sample-presentation-author:v0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=presentation-author"
```
