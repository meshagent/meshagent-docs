If you want to create a voice-tools service that will run in every room:
```
meshagent service create --name "voice-tools" --image docker.io/meshagent/sample-voice-tools:0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=voice-tools"
```

If you want to test the voice-tools with a single room:
```
meshagent service test --room my-room --name "voice-tools" --image docker.io/meshagent/sample-voice-tools:0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=voice-tools"
```
