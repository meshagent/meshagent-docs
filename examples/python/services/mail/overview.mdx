If you want to create a mail service that will run in every room:
```
meshagent service create --name "mail" --image docker.io/meshagent/sample-mail:0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=mail"
```

If you want to test the mail with a single room:
```
meshagent service test --room my-room --name "mail" --image docker.io/meshagent/sample-mail:0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=mail"
```
