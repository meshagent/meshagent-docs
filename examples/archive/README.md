# Archived Examples

This directory contains legacy MeshAgent examples that are kept for historical reference.

These examples are not part of the active docs set and may use older runtimes, commands, images, or patterns that are no longer recommended for new work.

Use the current examples under `examples/cli`, `examples/python`, `examples/typescript`, `examples/javascript`, `examples/dart`, and `examples/dotnet` when you need a supported starting point.

If you keep content here, treat it as archive-only and prefer linking to the current docs whenever there is a modern replacement.
