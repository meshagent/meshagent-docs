If you want to create a coder service that will run in every room:
```
meshagent service create --name "coder" --image docker.io/meshagent/sample-coder:v0.0.30 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=coder"
```

If you want to test the coder with a single room:
```
meshagent service test --room my-room --name "coder" --image docker.io/meshagent/sample-coder:v0.0.30 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=coder"
```
