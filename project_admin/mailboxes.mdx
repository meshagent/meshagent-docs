Mailboxes let MeshAgent receive inbound email and route it into a room workflow.

Most teams use mailboxes when they want an agent to handle email through a process mail channel or another queue-backed workflow.

## What a mailbox does

A mailbox maps an email address to a room and queue so inbound mail can become work for an agent or service.

## Where to manage mailboxes

- Use [MeshAgent Studio](../interfaces/meshagent_studio) for the primary UI flow.
- Use the CLI when you want quick setup or scripting.
- Use the [REST API](../rest_api/overview) when you need programmatic mailbox management.
