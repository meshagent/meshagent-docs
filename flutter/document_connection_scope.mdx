## Description

`DocumentConnectionScope` is a builder widget that manages the connection to a Mesh document. When initialized, it attempts to open the specified document at the provided `path` and then returns either an instance of `MeshDocument` or any resulting error through the builder callback. When the widget is disposed, it automatically closes the connection to the document.


## Example

```dart
import 'package:meshagent_flutter/meshagent_flutter.dart';

DocumentConnectionScope(
  path: '/some/path/to/document',
  room: room, // instance of MeshRoom
  builder: (context, document, error) {
    if (error != null) {
      // Handle the error
      return Text('Error: $error');
    }

    if (document == null) {
      // Document is still loading
      return Text('Loading...');
    }

    return Text('Document loaded: ${document.path}');
  })

```
