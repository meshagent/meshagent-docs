If you want to create a analyzer service that will run in every room:
```
meshagent service create --name "analyzer" --image docker.io/meshagent/sample-analyzer:v0.0.30 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=analyzer"
```

If you want to test the analyzer with a single room:
```
meshagent service test --room my-room --name "analyzer" --image docker.io/meshagent/sample-analyzer:v0.0.30 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=analyzer"
```
