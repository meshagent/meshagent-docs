If you want to create a voice-proofreader service that will run in every room:
```
meshagent service create --name "voice-proofreader" --image docker.io/meshagent/sample-voice-proofreader:0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=voice-proofreader"
```

If you want to test the voice-proofreader with a single room:
```
meshagent service test --room my-room --name "voice-proofreader" --image docker.io/meshagent/sample-voice-proofreader:0.0.31 --env="MESHAGENT_PORT=8090" --port "num=8090 type=meshagent.callable liveness=/ path=/agent participant_name=voice-proofreader"
```
